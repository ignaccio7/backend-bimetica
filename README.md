php artisan migrate

Ejecuta las migraciones pendientes (que no se han corrido aún)
NO borra datos existentes
Usa este cuando agregues nuevas migraciones y quieras mantener tus datos

php artisan migrate:rollback

Revierte la última "batch" de migraciones
Borra las tablas o cambios de esa batch
Útil para deshacer el último cambio

php artisan migrate:fresh

Elimina TODAS las tablas y ejecuta todas las migraciones desde cero
Es como empezar de nuevo completamente
⚠️ Pierdes todos los datos

php artisan migrate:refresh

Hace rollback de TODAS las migraciones y luego las vuelve a ejecutar
También pierdes todos los datos
Similar a fresh pero usando el método rollback

php artisan migrate:reset

Revierte todas las migraciones ejecutadas
Deja la base de datos vacía

Para crear enlace simbolico ya que estamos guardando imagenes en services
php artisan storage:link
